💳 पेमेंट गेटवे

सहभागींकडून ऑनलाइन पेमेंट स्वीकारण्यासाठी तुमचे Razorpay खाते कनेक्ट करा.

⚪ कनेक्ट नाही

Razorpay कॉन्फिगरेशन

तुमच्या API keys Razorpay Dashboard → Settings → API Keys वरून मिळवा. चाचणीसाठी Test keys वापरा आणि प्रत्यक्ष उत्पादनासाठी Live keys वापरा.

⚠️ तुमची Key Secret सर्व्हरवर सुरक्षितपणे साठवली जाते आणि कधीही फ्रंटएंडला दाखवली जात नाही.

1️⃣

वर तुमचा Razorpay Key ID आणि Secret टाका

2️⃣

3️⃣

नोंदणी दरम्यान सहभागी UPI/कार्ड/नेटबँकिंग द्वारे पेमेंट करतात
